# Welcome to MoirAI

  **MoirAI** is an AI-powered storytelling platform that brings the magic of tabletop RPGs, immersive narratives, and creative interactions to life. Whether you're adventuring as
  a player or crafting stories as a director, MoirAI adapts to your desires and makes every experience uniquely yours.

  This GitHub organization houses the repositories for MoirAI and its modules. Here, you'll find everything from the core application to experimental features, utilities, and
  community-driven contributions.

  ## What you'll find here

  ### Core Repositories

  - **[MoirAI Backend](https://github.com/moirairpg/story-engine):** Built with Java, processes the logic and game systems for MoirAI.
  - **[MoirAI Frontend](https://github.com/moirairpg/moirai-ui):** MoirAI's UI built with React, for managing resources in the game.

  ## Contributing

  1. Fork the repository you're interested in.
  2. Submit your changes via a pull request.
  3. Our developers will review and, if approved, merge your contributions!

  Check out the individual repository README files for guidelines and more information.
